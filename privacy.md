# Политика конфиденциальности приложения PinGen MAXYS

**Дата вступления в силу:** 2024

## 1. Общие положения
Настоящая Политика конфиденциальности описывает, как приложение PinGen MAXYS (далее — «Приложение») собирает, использует и раскрывает информацию о пользователях при использовании Приложения и интеграции с Pinterest API.

## 2. Сбор и использование данных
Приложение запрашивает доступ к Pinterest API исключительно для:
- Создания и публикации пинов от имени пользователя
- Управления контентом на досках Pinterest
- Получения базовой информации о профиле пользователя

## 3. Какие данные собираются
- Токены доступа Pinterest (хранятся локально на устройстве пользователя)
- Информация о профиле (имя, ID пользователя)
- Данные о созданных пинах

## 4. Хранение данных
Все данные хранятся только на устройстве пользователя. Приложение не передает данные третьим лицам.

## 5. Безопасность
Мы принимаем разумные меры для защиты данных пользователей от несанкционированного доступа.
